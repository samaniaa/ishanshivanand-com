Colors · semantic tokens

TokenLightDusk ([data-theme='dusk'])

Components consume only these semantic tokens. On the live site the day engine flips them via body[data-phase='dark'] (light-on-dark ink #f7f3e8) in lockstep with the sky, so ink and sky can never disagree. [data-theme='dusk'] shown here is the equivalent static theme (ink #fffcf6) — same intent, a fixed swatch rather than the scrubbed flip.
